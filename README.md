# Gurumusic

Gurumusic is a React, Vite, and Tailwind CSS music application for streaming curated playlists, discovering independent uploads, and tracking artist performance through an analytics dashboard.

## Features

- Browse Guruplay playlists, artists, and spotlight uploads.
- Require authentication before listening to songs.
- Use reusable popup authentication for login and signup.
- Collect signup profile details including date of birth and gender.
- Manage uploaded music and view dashboard metrics.

## Development

Install dependencies and start the local development server:

```sh
npm install
npm run dev
```

Create a production build:

```sh
npm run build
```

Run linting:

```sh
npm run lint
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase client
